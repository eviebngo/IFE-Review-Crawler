# Terms of Service — IFE ReviewDB

IFE ReviewDB is a personal, non-commercial research tool that indexes
publicly available airline in-flight-entertainment reviews (YouTube
videos and press articles) for personal analysis. These terms apply
to that tool and its use of YouTube API Services.

## Use of the Service

- This project is provided as-is, for personal research purposes,
  with no warranty of any kind.
- There is no public sign-up, account system, or user-facing service —
  it runs locally and via a scheduled job under the owner's own
  Google Cloud project and API credentials.
- No YouTube video content is downloaded, redistributed, rehosted, or
  modified. The tool stores only public metadata (titles,
  descriptions, view counts, publish dates) and, where available,
  publicly accessible caption transcripts, and always links back to
  the original video on youtube.com.

## Compliance

- Use of YouTube API Services through this project is subject to the
