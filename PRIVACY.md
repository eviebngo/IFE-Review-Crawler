# Privacy Policy — IFE ReviewDB

IFE ReviewDB is a personal, non-commercial research tool that indexes
publicly available airline in-flight-entertainment reviews (YouTube
videos and press articles) for personal analysis.

- This tool does not collect, store, or share any personal data about
  its user (Evie Ngo) or about any third party.
- It does not require sign-in, does not use OAuth, and does not access
  any private YouTube account data.
- It reads only publicly available YouTube metadata (video titles,
  descriptions, view counts, publish dates) and publicly available
  caption transcripts via the YouTube Data API v3, using a server-side
  API key.
- No data collected by this tool is sold, shared, or used for
  advertising. It is stored locally / in this repository for personal
  research purposes only.
